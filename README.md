# blogpost-frontend-challenge

## Running this project locally

```bash
# with yarn
git clone git@github.com:brunonasc74/blogpost-frontend-challenge.git
cd .\blogpost-frontend-challenge
yarn
yarn dev

# with npm
git clone git@github.com:brunonasc74/blogpost-frontend-challenge.git
cd .\blogpost-frontend-challenge
npm install
npm run dev
```
